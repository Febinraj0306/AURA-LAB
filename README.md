## Overview
A dark, premium, single-page landing page built for a luxury AI-powered web design agency. It features a cinematic, editorial aesthetic with custom liquid glassmorphism effects, dynamic video backgrounds, and high-performance entrance animations.

## Tech Stack
- **Framework**: React 19 + Vite
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion (`motion/react`)
- **Icons**: Lucide React
- **Video Delivery**: HLS.js (for adaptive video streaming)

## Features
- **Cinematic Hero**: Full-screen looping video background with staggered, animated typography.
- **Liquid Glass Design**: Custom CSS utility classes for a premium, translucent aesthetic with layered gradients and drop shadows.
- **Dynamic Content**: IntersectionObserver-triggered scroll animations for text and interactive elements.
- **Adaptive Video Backgrounds**: High-performance HLS streaming video backgrounds, including desaturated filters for visual hierarchy.
- **Responsive Architecture**: Fluid design system that scales seamlessly from mobile devices to ultra-wide desktop screens.

## Getting Started

### Prerequisites
- Node.js (v18 or higher recommended)
- npm, yarn, or pnpm

### Installation
1. Clone the repository to your local machine.
2. Install dependencies:
```bash
npm install
```
3. Start the Vite development server:
```bash
npm run dev
```
4. Open the provided `localhost` URL in your browser to view the application.

## Styling Architecture
The project utilizes Tailwind CSS v4 integrated with custom CSS properties in HSL format for a strictly controlled luxury palette (focusing on rich blacks, crisp whites, and subtle muted tones). 

Typography is driven by Google Fonts:
- **Headings**: Instrument Serif (Italic)
- **Body**: Barlow (Weights 300, 400, 500, 600)

## Animation Patterns
The site employs `motion/react` for smooth, cinematic motion:
- Staggered word-by-word blur-in reveals for primary headings.
- Subtle `scale-105` hover transitions for calls to action.
- Coordinated fade-ins on scroll to maintain a sophisticated pace.
